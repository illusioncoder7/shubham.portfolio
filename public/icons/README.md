# Nav icons

The top navigation uses **SVG icons with transparent backgrounds** (recommended):

- `nav-top.svg` – Home
- `nav-experience.svg` – Experience
- `nav-tech-stacks.svg` – My Skills
- `nav-education.svg` – Education
- `nav-contact.svg` – Contact

These are vector icons in the site accent color (`#41ccc0`), scale cleanly, and have no background. Display size is controlled by CSS (max 44×44px).

Optional: if you add PNG versions (e.g. `nav-top.png`), you can switch the app to use `.png` for fallback; the app currently loads the `.svg` files above.
